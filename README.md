# 📖 QURAN'S HOME

**QURAN'S HOME** هو موقع بسيط يهدف إلى عرض **القرآن الكريم** على شكل صفحات يمكن تصفحها بسهولة.

الموقع ما زال مشروعًا بسيطًا وفي مرحلة التطوير.

---

## 🌙 فكرة الموقع

فكرة الموقع هي إنشاء صفحة على الإنترنت يمكن من خلالها قراءة **القرآن الكريم** بطريقة سهلة ومنظمة.

الهدف هو توفير مكان بسيط لقراءة القرآن من المتصفح دون تعقيد.
